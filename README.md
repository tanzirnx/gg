# NITHA AI Assistant

A modern Android AI voice assistant powered by OpenRouter API.

## Features
- Voice Commands & AI Chat
- Device Control (Apps, Settings, Flashlight)
- File Management
- Notification Intelligence
- OCR & Vision
- Automation Engine
- Multi-language Support

## Build Instructions
1. Open in Android Studio
2. Sync Gradle
3. Build APK

## GitHub Actions
Auto-builds APK on push to main branch.
